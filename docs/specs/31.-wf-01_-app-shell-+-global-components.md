# 1. App Shell Frame
The App Shell is the outermost UI frame. It is rendered ONCE at application load and persists across all navigation. Workspace content renders INSIDE the App Shell's content area. The App Shell owns: System Bar, Navigation Rail, Patient Banner, and Content Area frame.
